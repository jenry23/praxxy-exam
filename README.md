## Installation

```bash
$ git clone https://github.com/jenry23/praxxy-exam.git
$ cd praxxy-exam/
$ composer install
$ cp .env.example .env
update .env and set your database credentials
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
$ npm install
$ npm run dev
$ php artisan serve

```
